About

This app is able to scrape the latest apartment listings from Craigslist to help you find your next home.

It also calculates commute times to your workplace using the Google Maps Distance Matrix API.

Simply input your search criteria and preferences, and let the program do the rest!

This project was created by Andrew Kim at Georgetown University.
